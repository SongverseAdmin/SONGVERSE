# 大风云扬：中国音乐出海服务体系标准
# Da Feng Yun Yang: China Music Global Service System Standard

**版本 / Version:** Draft v0.1  
**状态 / Status:** 开源实践版 / Open-source Practice Version  
**发布源 / Canonical Source:** 本 GitHub 仓库为本标准的权威发布源  
**许可 / License:** 待定（建议未来添加 LICENSE 文件）

大风云扬(中国音乐出海服务体系标准)是一个面向中国音乐内容“全球发行、跨文化推广、数据与流程规范”的开放式行业参考框架。  
大风云扬(中国音乐出海服务体系标准)为文档体系名称，不代表商标、企业、组织或任何法律主体。

Da Feng Yun Yang is an open, voluntary documentation framework for global distribution and cross-cultural promotion of music originating from China.  
The term "Da Feng Yun Yang" does not represent a trademark, organization, company, or legal entity.

## 1. 定位 / Positioning

**中文：**  
大风云扬 为中国音乐人、创作者、团队及从业者提供一套可操作、可复制、可审计的音乐出海流程，包括：  
- 全球发行  
- 跨文化推广  
- 数据规范  
- 生命周期运营  

**English:**  
Da Feng Yun Yang provides a standardized, repeatable, and auditable workflow for Chinese musicians and teams to distribute and promote music internationally, with Europe and North America as primary reference markets while remaining globally applicable.

## 4. 标准流程 / Standard Process

大风云扬 遵循欧美音乐行业常用的“生命周期工作流（Lifecycle Workflow）”，覆盖从准备、发行、推广到持续运营的全流程。

Da Feng Yun Yang follows a lifecycle workflow widely used in the EU/NA music industry.

---

### PREPARE — 全球发行准备 / Global Release Preparation

中文：
- 确定版本：Master / Radio Edit / Instrumental  
- 准备国际标准元数据（ISRC、UPC、Credits）  
- DSP 标准封面（3000×3000）  
- 音频、封面、Press Kit 等全套素材  
- 明确目标市场与受众  
- 制定发行计划（Release Plan）

English:
- Define track versions  
- Prepare global metadata  
- Prepare DSP-compliant artwork  
- Organize audio/visual/press assets  
- Identify target markets and audiences  
- Build release plan

**Deliverables:** Metadata Package / Assets / Release Plan

---

### DISTRIBUTE — 全球数字分发 / Global Digital Distribution

中文：
- 按标准格式交付 WAV + Cover + Metadata  
- 全球 DSP 分发  
- Pre-save / Pre-add 设置  
- 时间同步（跨时区）  
- DSP QC 跟进  

English:
- Deliver WAV + artwork + metadata  
- Global DSP distribution  
- Pre-save / Pre-add activations  
- Time-zone aligned release  
- DSP QC monitoring  

**Deliverables:** DSP Confirmation / Pre-save Link / QC Checklist

### 5. Radio Promotion（欧美电台推广）

中文：  
欧美电台推广适用于 Pop / Rock / Dance / R&B / AC / Alternative 等类型，是获得行业可信度和大众曝光的重要渠道。

流程包含：  
- 选择电台格式（Top 40 / Hot AC / College / Dance）  
- 准备 Radio Edit（2:50–3:30，Clean Version）  
- 准备 Radio Servicing Package  
- 联系 Music Director / Program Director  
- 跟踪电台播放（Spins）与名单新增（Adds）

English:  
Radio promotion is a key component in many EU/NA genres.

Workflow:  
- Select radio format  
- Prepare Radio Edit (2:50–3:30, clean)  
- Build servicing package  
- Reach MD/PD  
- Track spins & adds  

**Deliverables:** Editorial Pitch / Content Package / Press Kit / Radio Servicing Pack / Spin Report
